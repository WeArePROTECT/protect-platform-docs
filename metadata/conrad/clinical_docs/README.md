# Clinical Data Handling Documentation

This directory contains documentation describing how Conrad clinical metadata is structured and handled for sharing within PROTECT.

## Contents

These documents describe:
- How clinical metadata is organized and structured
- Decisions about what data is included/excluded
- Handling conventions for clinical data sharing

## Included Documents

- **`clinical_data_handling_decisions.docx`** — Clinical data handling decisions document 
- **`clinical_data_handling_decisions_table.docx`** — Clinical data handling decisions (table format)
- **`clinical_data_handling_decisions_updated.docx`** — Updated version of clinical data handling decisions

## Authoritative Version

clinical_data_handling_decisions_updated.docx is the authoritative version.
